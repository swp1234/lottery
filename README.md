# 행운의 번호 생성기 🍀

로또 6/45, 연금복권 번호를 무작위로 생성하는 PWA 앱입니다.

## 기능

- 🎱 **로또 6/45** - 1~45 중 6개 번호 자동 생성
- 💰 **연금복권** - 1~5조, 6자리 번호 생성
- ⭐ **여러 세트 생성** - 한 번에 최대 10개 세트
- 💾 **저장 기능** - 마음에 드는 번호 저장
- 📊 **통계** - 생성 횟수, 행운의 요일 분석
- 🌓 **다크/라이트 모드** - 눈에 편안한 테마
- 📱 **PWA 지원** - 모바일 홈 화면에 설치 가능

## 기술 스택

- HTML5, CSS3, Vanilla JavaScript
- Progressive Web App (PWA)
- LocalStorage for data persistence
- Responsive Design

## 로컬 실행

```bash
# 간단한 HTTP 서버 실행
python -m http.server 8000

# 브라우저에서 접속
http://localhost:8000
```
